# test-task-Indev

* Node.js
* Nest.js
* TypeScript
* PostgreSQL & TypeORM
* JWT
* Swagger

## For starting project:

<code>npm run start:prod</code>

### ! Env files specially added to the repo (you just need to set your own Postgres DB)

* Registration
* Password recovery by mail
